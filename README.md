# CodSoft Python Programming Internship

This repository contains the projects completed as part of the CodSoft Python Programming Internship.

## Completed Tasks

Task 1 - Rule-Based Chatbot
A professional GUI-based chatbot developed using Python and Tkinter. It responds to user queries using predefined rules and pattern matching techniques.

Task 2 - Tic Tac Toe Game
A two-player Tic Tac Toe game developed using Python. The game provides an interactive interface, validates moves, and automatically detects the winner or a draw.

Task 3 - Recommendation System
A recommendation system developed using Python that suggests items based on user preferences using predefined recommendation logic.

Task 4 - Face Detection
A real-time face detection application built using Python and OpenCV. It detects human faces from images or webcam video using Haar Cascade classifiers.

***Technologies Used

- Python
- Tkinter
- OpenCV
- NumPy
- Regular Expressions
